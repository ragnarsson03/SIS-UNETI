# SIS-UNETI (Sistema de Información Estudiantil)

### 🚀 Cómo iniciar el entorno de desarrollo
1. Clona el repositorio.
2. Copia `.env.example` a un nuevo archivo `.env`.
3. Asegúrate de tener **Docker Desktop** instalado.
4. Ejecuta: `docker compose up --build`
5. La API estará en: `http://localhost:3000`
6. La DB está protegida internamente en el puerto 5432.
